# Ende {#end}

Die **Ende** -Aktivität **Flusssteuerung** Aktivität. Mit dieser Aktivität können Sie das Ende eines Workflows grafisch markieren. Diese Aktivität hat keine funktionalen Auswirkungen und ist daher optional.
